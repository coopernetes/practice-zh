practice-zh

This is a web app that I created to help myself study Chinese (specifically 官话). I use it to supplement my learning on my phone using HelloChinese & Pleco. Unfortunately, neither of those apps have any sort of desktop client so I wanted the ability to test myself on a laptop or desktop computer.

The main features I plan to add are:

  • Phrase and sentence-based quizzes. I already get lots of practice on the apps with individual character quizzes which I feel is insufficient.
  • Create a flash card system with spaced repetition
  • Track progress and performance over time

Some stretch goals include integrating Google's APIs for text-to-speech so you can hear both question and answer phrases.

I have little-to-no interest in adding AI to this project. What I will likely use are LLM chat bots, such as ChatGPT & Gemini, to generate phrases to use as the test data. My goal is to be able to generate 1000's of phrases, store them in a database and ensure they are grammatically correct while allowing phrases to be more "dynamic" (for example, using different adjectives on specific subject/noun combinations). Because this is meant to be used for learning, I am prioritizing accuracy over variety (for now).

This project is open source so you can find the code on GitHub.
